# @am/rc-component

A common React component library

## How to use

Install the library

```
npm i --save @am/rc-component
```

Import library

```javascript
import { Tab, Accordion } from "@am/rc-component";
```

## Storybook

To run the storybook

```
npm run storybook
```

## Component

The list of common component:

- Tab
- Accordion
